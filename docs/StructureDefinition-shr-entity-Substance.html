Substance

 

The official URL for this profile is: 

http://hl7.org/fhir/us/fhirURL/StructureDefinition/shr-entity-Substance

Any matter of defined composition that has discrete existence, whose origin may be biological, mineral or chemical.

This profile builds on Substance.

This profile was published on Thu Feb 07 00:00:00 EST 2019 as a draft by The MITRE Corporation.

  _Entry.EntryId maps to id
  Language maps to language
  Narrative maps to text
  ImplicitRules maps to implicitRules
  Metadata.VersionId maps to meta.versionId
  Metadata.LastUpdated maps to meta.lastUpdated
  Metadata.SecurityLabel maps to meta.security
  Metadata.Tag maps to meta.tag
  Identifier maps to identifier
  Status maps to extension
  Type maps to code
  Ingredient maps to ingredient
  Ingredient.IngredientAmount maps to ingredient.quantity
  Ingredient.SubstanceOrCode.Substance maps to ingredient.substance
