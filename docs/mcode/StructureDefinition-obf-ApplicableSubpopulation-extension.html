ApplicableSubpopulation

 

The official URL for this extension is: 

http://mcodeinitiative.org/us/mcode/StructureDefinition/obf-ApplicableSubpopulation-extension

Codes to indicate the target population this reference range applies to. For example, a reference range may be based on the normal population or a particular sex or race.

This extension was published on Wed Oct 23 00:00:00 EDT 2019 as a draft by The MITRE Corporation.
