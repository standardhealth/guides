HeadlessLaboratoryPanel

 

The official URL for this profile is: 

http://hl7.org/fhir/us/shr/StructureDefinition/shr-core-HeadlessLaboratoryPanel

A panel that does not have an overall value that summarizes the entire panel.

This profile builds on http://fhir.org/guides/argonaut/StructureDefinition/argo-observationresults.

This profile was published on Mon Apr 08 00:00:00 EDT 2019 as a draft by The MITRE Corporation.
