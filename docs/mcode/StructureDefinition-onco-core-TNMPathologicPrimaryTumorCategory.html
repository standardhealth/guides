TNMPathologicPrimaryTumorCategory

 

The official URL for this profile is: 

http://mcodeinitiative.org/us/mcode/StructureDefinition/onco-core-TNMPathologicPrimaryTumorCategory

Category describing the primary tumor, based on its size and extent, assessed through pathologic analysis of a tumor specimen.

This profile builds on Observation.

This profile was published on Wed Oct 23 00:00:00 EDT 2019 as a draft by The MITRE Corporation.
