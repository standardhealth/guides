SurgicalProcedure

 

The official URL for this profile is: 

http://mcodeinitiative.org/us/mcode/StructureDefinition/obf-SurgicalProcedure

A procedure that involves a physical intervention on tissues.

Conformance note: If an ICD-10-PCS code is used in the code attribute, and an equivalent SNOMED CT is available, the resulting Procedure instance will not be compliant with US Core Profiles.

This profile builds on http://fhir.org/guides/argonaut/StructureDefinition/argo-procedure.

This profile was published on Wed Oct 23 00:00:00 EDT 2019 as a draft by The MITRE Corporation.
