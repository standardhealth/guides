<p><b>PrimaryCancerDisorderVS ValueSet</b></p>
<p>Types of primary malignant neoplastic disease, coded in SNOMED CT, ICD-10-CM, or ICD-O-3.
*SNOMED CT coding: Use a code from the disorder hierarchy under SNOMED 363346000 'Malignant neoplastic disease (disorder)' exluding codes descending from SNOMED CT 128462008 'Secondary malignant neoplastic disease (disorder)'. If body site is not precoordinated (i.e., implied by the code), the site of the primary tumor should be specified in the body site attribute.
* ICD-10-CM coding: Use one of the codes given in this value set representing primary malignant neoplasms and neoplasms of uncertain or unspecified behavior.
* ICD-O-3 specifies morphology and topography, not disorder; mCODE requires in this case that the disorder code must be SNOMED-CT 363346000  'Malignant neoplastic disease (disorder)'. The ICD-O-3 histology/morphology/behavior code and the topography code should then be specified in ICD-O-3 codes in the fields HistologyMorphologyBehavior extension and bodySite, respectively.</p>
