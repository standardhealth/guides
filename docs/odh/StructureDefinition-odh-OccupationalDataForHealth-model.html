OccupationalDataForHealth

 

The official URL for this logical model is: 

http://hl7.org/fhir/us/odh/StructureDefinition/odh-OccupationalDataForHealth-model

Information about a person’s work and volunteer positions to support patient care, population health and public health activities.. The profile is based on FHIR Composition that contains a single section, OccupationalDataSection.

This logical model was published on Wed May 08 00:00:00 EDT 2019 as a draft by HL7.
