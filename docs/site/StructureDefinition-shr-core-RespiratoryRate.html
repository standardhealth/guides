RespiratoryRate

 

The official URL for this profile is: 

http://hl7.org/fhir/us/shr/StructureDefinition/shr-core-RespiratoryRate

The rate of breathing (inhalation and exhalation) measured within in a unit time, expressed as breaths per minute.

This profile builds on Observation.

This profile was published on Wed May 01 00:00:00 EDT 2019 as a draft by The MITRE Corporation.
